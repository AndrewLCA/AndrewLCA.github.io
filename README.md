# 🌐 Andrew Lu — Personal Website

A fully custom-designed personal website built to showcase my background, research, and projects through immersive, themed experiences.

## 🚀 Live Site
👉 https://andrewlca.github.io

---

## 📌 Overview

This is a multi-page portfolio website built from scratch using HTML, CSS, and JavaScript. Each page has a distinct theme to reflect different aspects of my personality, work, and interests.

The goal of this project was to combine **creative UI design** with **technical implementation**, resulting in an interactive and visually engaging experience rather than a traditional static portfolio.

---

## ✨ Features

- 🎨 Fully custom UI (no templates used)
- 🌸 Animated landing page with smooth transitions
- 🎮 Unique themed subpages
- 💻 Interactive terminal-style research page
- ☕ Lofi-style contact page with animated cards
- 📱 Responsive design for different screen sizes
- ⚡ Lightweight and fast (pure HTML/CSS/JS)
